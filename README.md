# Nova Assistant

Nova Assistant is a desktop-based AI voice assistant built using Python. It integrates speech recognition, text-to-speech, and AI-based natural language processing to perform user commands and automate tasks.

## Features

- Responds to voice commands
- Opens applications and websites
- Provides information through web searches
- Converts text to speech
- Handles conversations using AI models
- Supports task automation

## Tech Stack

- **Language:** Python  
- **Libraries:**
  - SpeechRecognition
  - Pyttsx3
  - PyAutoGUI
  - Transformers (Hugging Face)
  - Torch
  - OpenCV
  - Other standard Python modules (os, datetime, webbrowser, subprocess, etc.)
